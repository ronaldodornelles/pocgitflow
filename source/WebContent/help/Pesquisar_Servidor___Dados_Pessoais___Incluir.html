Descrição:

 

Esta é a tela de inclusão de Servidor.

 

Como incluir um Servidor no sistema?

 

Preencha as informações de Dados Pessoais:

 

1. Para vincular uma pessoa ao cadastro de servidor, selecione a opção Vincular(1) para buscar a pessoa desejada.

2. É obrigatório o preenchimento do campo Matrícula, preenchendo com a matrícula do Servidor.

3. É obrigatório o preenchimento do campo Identificação única, preenchendo com a identificação única do Servidor.

4. Preencha o campo E-mail Institucional, preenchendo com o e-mail institucional do Servidor.

 

5. Clique na aba Dados Funcionais (2) para continuar a inclusão do servidor.

6. Clique na aba Cargo/Função (3) para continuar a inclusão do servidor.

 

Importante: É obrigatório o preenchimento de alguns campos nas abas Dados Funcionais e Cargo/Função.

 

Ponto(s) de Extensão:

 

1. Caso você queira voltar à tela de Pesquisa de servidor selecione o botão Voltar (4) e você será direcionado à tela de pesquisa.
