Descrição:

 

Esta é a tela de visualização do(s) registro(s) de Servidor(s). A partir dela é possível realizar a consulta (através do botão Consultar) de servidores cadastradas no sistema:

 

Como consultar os registros de servidores existentes no sistema?

 

1. Selecione o botão Consultar (6). *

2. Você será direcionado à tela de listagem de servidor(s) e terá acesso ao(s) registro(s) de servidor(s) cadastrado(s).

 

Ponto(s) de Extensão:

 

1. Caso você queira vincular um servidor selecione o botão Vincular (8) para vincular um servidor.

2. Caso você queira voltar a tela de edição de Unidade Organizacional selecione o botão Cancelar (7).

 

Observações:

 

      * Caso você queira realizar uma consulta específica, utilize os filtros de pesquisa Nome Funcionário(1), Matrícula(2), CPF(3), Data entrada IPHAN(4) e Situação Funcional(5). Você pode utilizar quantos e quais filtros desejar para realizar a consulta.
