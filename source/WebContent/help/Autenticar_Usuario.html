Descrição:

 

Esta é a tela de login do sistema. A partir dela é possível realizar o login (através do botão Login) para ter acesso ao sistema.

 

Como efetuar o login no sistema?

 

1. é de preenchimento obrigatório o campo Usuário(1), preenchendo com o nome de usuário de acesso ao sistema.

2. é de preenchimento obrigatório o campo Senha(2), preenchendo com a senha do usuário de acesso ao sistema.

3. Acione o botão Login(3) para acessar o sistema.

 

Ponto(s) de Extensão:

 

1. Selecione o campo Lembrar-me(4) para manter o nome de usuário no campo Usuário(1) para um próximo acesso ao sistema.
