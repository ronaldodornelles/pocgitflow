Descrição:

 

Esta é a tela de edição dos dados pessoais do servidor. *

 

* Todos os campos são de somente leitura, não sendo possível modificar qualquer campo.

 

1. Caso você queira editar Dados Funcionais do servidor selecione a aba Dados Funcionais (2).

 

2. Caso você queira editar Cargo/Função do servidor selecione a aba Cargo/Função (3).

 

3. Caso você queira editar Afastamento/Ocorrência do servidor selecione a aba Afastamento/Ocorrência (4).

 

4. Caso você queira editar Cessão/Requisição do servidor selecione a aba Cessão/Requisição(5).

 

5. Caso você queira editar Formação Acadêmica do servidor selecione a aba Formação Acadêmica (6).

 

6. Caso você queira editar Dependentes do servidor selecione a aba Dependentes (7).

 

Ponto(s) de Extensão:

 

1. Caso você queira voltar à tela de Pesquisa de servidor selecione o botão Voltar(1) e você será direcionado à tela de pesquisa.
