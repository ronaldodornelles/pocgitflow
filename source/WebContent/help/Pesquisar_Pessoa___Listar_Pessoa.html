Descrição:

 

Esta é a tela de visualização do(s) registro(s) de pessoa(s). A partir dela é possível realizar a consulta (através do botão Consultar) de pessoas cadastradas no sistema:

 

Como consultar os registros de pessoas existentes no sistema?

 

1. Selecione o botão Consultar (3). *

2. Você será direcionado à tela de listagem de pessoa(s) e terá acesso ao(s) registro(s) de pessoa(s) cadastradas.

 

Ponto(s) de Extensão:

 

1. Caso você queira vincular uma pessoa ao cadastro de servidor selecione o botão vincular (5) para vincular uma pessoa.

2. Caso você queira voltar a tela de cadastro de servidor selecione o botão cancelar (4) para voltar a tela de inclusão de servidor.

 

Observações:

 

      * Caso você queira realizar uma consulta específica, utilize os filtros de pesquisa Nome (1) e CPF (2). Você pode utilizar quantos e quais filtros desejar para realizar a consulta.
